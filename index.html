Here are some of my projects

  • Secret Santa Selections Manager - Manages Secret Santa selections. Makes selections considering exceptions, and then alerts participants of who they are assigned to give a gift to via email.
  • Spotify Playlist Generator - Takes a text-based list of songs and puts it in a Spotify playlist.
  • March Madness Predictor - Used for creating a bracket for the Men's NCAA Division 1 Basketball Tournament. Offers an evaluation and prediction mode. Evaluation mode is a manual bracket selection process that provides relevant information to a user when making selections, such as rankings, recent game results, etc. Prediction mode makes automated selections using various prediction methods.
  • Ticket to Ride Graph Analysis - Basic analysis of the board game Ticket to Ride, modelling the board as an undirected graph in Rust.
  • Grocery List Creator - Takes a list of recipes and generates a list of ingredients for grocery shopping. Uses a very limited open source API, and definitely needs some improvements to be useful, but it's a fun concept that I'd love to expand on some more.
